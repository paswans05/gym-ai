# GymAI

**Camera-free AI gym trainer powered by Wi-Fi signals**

GymAI detects and corrects workout form (squat, push-up, deadlift) using **Wi-Fi signal sensing** instead of cameras.  
It is privacy-first, works in low light, and is designed for smart gyms and home workouts.
